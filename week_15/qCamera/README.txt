qCamera Main Window

Camera GUI Window created using PyQt5. 

Gifgram was going to have a feature using webcam to create images and modify them.
Also record video from user's webcam to create gifs and modify them as well.

Original UI files were converted to Python and Original C++ code created from qtcreator converted as well.
